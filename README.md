# 🚀 Formação em Lógica de Programação

Bem-vindo(a) ao meu repositório de estudos! Este espaço é dedicado ao aprendizado dos fundamentos da programação, onde documento minha evolução e a resolução de desafios lógicos.

Acredito que uma base sólida é o diferencial de qualquer pessoa desenvolvedora. Por isso, este repositório foca em transformar problemas complexos em soluções estruturadas.

---

## 🧠 Conceitos Aplicados

Neste projeto, você encontrará implementações que utilizam os pilares essenciais da programação:

* **Variáveis:** Armazenamento e manipulação de dados de forma eficiente.
* **Operadores:** Uso de lógica aritmética, relacional e lógica booleana para processamento de informações.
* **Estruturas de Decisão:** Implementação de fluxos condicionais (`if/else`, `switch/case`) para controle de cenários.
* **Laços de Repetição:** Automação de tarefas e iteração sobre dados utilizando `for`, `while` ou `do-while`.

---

## 🛠️ O que estou construindo?

Os códigos aqui presentes refletem meu progresso em:
1.  **Entender o problema** (Input)
2.  **Processar a informação** (Lógica/Algoritmo)
3.  **Entregar o resultado** (Output)

---

## 📁 Estrutura de Pastas

* `Desafio - Classificador de Nível de Herói`: Desafio inicial para colocar em prática o uso de variáveis, operadores, estruturas de decisão e laços de repetição.
